# Flightning Utils

This subpackage contains additional simulation tools for quadrotor simulation.

| Module                          | Description                                    |
|---------------------------------|------------------------------------------------|
| evaluation.py                   | tools to evaluation rollouts                   |
| math.py                         | math functions                                 |
| plotting.py                     | plotting functions for rollouts                |
| printing.py                     | printing functions for rollout evaluations     |
| pytrees.py                      | pytree utils                                   |
| spaces.py                       | space utils for env wrappers                   |
| trajectory.py                   | trajectory functions (e.g. analytic min snap)  |
