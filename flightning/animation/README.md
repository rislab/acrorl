# Animation

This directory contains the animation tools for Flightning.

Adapted from Kanishke Gamagedara's [pyplot-3d](https://github.com/kanishkegb/pyplot-3d)

| Module                 | Description                                                        |
|------------------------|--------------------------------------------------------------------|
| basic.py               | 3D primitive shapes                                                |
| uav.py                 | 3D uav primitive                                                   |
| viz.py                 | Main QuadrotorAnimation class for rendering, displaying, & saving  |
