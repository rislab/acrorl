# Flightning Scripts

This subpackage contains scripts for training, evaluting, and exporting policies.

| Script                          | Description                                                             |
|---------------------------------|-------------------------------------------------------------------------|
| eval_control.py                 | Evaluation script for hopf control                                      |
| eval_learned_control.py         | Evaluation script for learned reference modulation + hopf control       |
| export.py                       | Script to export checkpoint to onnx model for hardware deployment       | 
| train.py                        | Training script for inversion reference modulation                      |
